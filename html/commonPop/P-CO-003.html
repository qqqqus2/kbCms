모니터링 파일 업로드
파일 업로드 *

업로드 파일은 csv파일만 가능하며 10MB까지 등록할 수 있습니다.

업로드한 파일명이 출력됩니다(다운로드 가능)1.jpg(1542byte)

다른 업로드한 파일명123이 출력됩니다(다운로드 가능)2.jpg(1542byte)

또 다른 업로드한 파일명123이 출력됩니다(다운로드 가능)3.jpg(1542byte)

업로드 파일 내용 예시 #업로드 파일 내용 예시

유의사항

  • 업로드 파일은 업로드 파일 내용 예시를 참고하여 형식에 맞게 작성하세요 (데이터 구분자 : ‘|’, 건마다 줄바꿈)
  • 문서 보안(DRM)을 해제한 영문파일명의 csv파일을 업로드하세요.
  • csv파일 업로드 후 [등록]버튼을 클릭하시고 반드시 자료 등록 결과(오류 발생 시, 해당 행/열 값 확인 가능)를 확인하세요.
  • 오류가 발생한 경우 업로드한 파일의 데이터 전체가 등록되지 않습니다.
