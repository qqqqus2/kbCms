통계지표 도움말
  • 조회수 콘텐츠 상세화면에 접속한 수로, 동일 고객의 중복 조회를 포함합니다.
  • 조회고객수 콘텐츠에 접속한 고객수로, 1개 이상의 콘텐츠를 조회한 고객수로 카운트합니다.
  • 긍정수 오늘의 콕 콘텐츠의 '추천해요' 수를 표기합니다.
    현재 프라임클럽 콘텐츠의 긍정수는 제공하지 않습니다.
  • 부정수 오늘의 콕 콘텐츠의 '아쉬워요' 수를 표기합니다.
    현재 프라임클럽 콘텐츠의 부정수는 제공하지 않습니다.
  • 유지율 전월에 이어 연속 방문한 고객 비율을 표기합니다.
    * 유지율(월) = (당월방문고객수-신규고객수) / 전월방문고객수 X 100
  • 조회고객수
    (연관종목)     연관종목을 포함한 콘텐츠를 조회한 고객수를 표기합니다.
  • 거래고객수 연관종목을 포함한 콘텐츠를 조회한 고객 중 해당 종목을 거래한 거래 고객수를 표기합니다.
  • 거래전환율 연관종목을 포함한 콘텐츠를 조회한 고객 중 해당 종목을 거래한 거래 고객 비율을 표기합니다.
    * 거래전환율 : 거래고객수/조회고객수(연관종목) X 100
  • 성과거래금액 연관종목을 포함한 콘텐츠를 조회하고 일주일 이내 해당 종목을 거래한 금액을 표기합니다.
    * 국내주식/ETF, 해외주식/ETF : 매도약정금액 + 매수약정금액
  • 성과수수료 연관종목을 포함한 콘텐츠를 조회하고 일주일 이내 해당 종목의 거래로 발생한 수수료를 표기합니다.
    * 국내주식/ETF, 해외주식/ETF : 매도수수료 + 매수수수료
